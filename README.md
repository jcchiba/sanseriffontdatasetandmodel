# San rerif Font StarGAN Model and Dataset.


Dataset and resulting model of san-serif font 96 basic characters trained using StarGAN.
Dataset contains 22,025  images of Font Glyph Characters divided in 96 categories corresponding to Utf-8 character encoding.
